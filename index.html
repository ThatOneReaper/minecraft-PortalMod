Initial Commit31 Jul 2015

So begins the project. We set out to design a minecraft mod to practice and test coding and design skills. We're starting something simple. A static portals mod for Minecraft 1.8. Until we become a little more creative in the naming, we'll be calling the project PortalMod. We will be using the latest recommended version of forge.

Our first order of business is setting up our workspaces, and starting some simple code. Adding a block, and simple transposition of an entity. Also, we now have a webpage for the project! (That's this one)

And we're on our way.
